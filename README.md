# ♟️ Java Chess Game

## 📌 Overview

This project is a **Java-based chess game** that implements the core logic of chess, including piece movement, game rules, and state management.

It was developed as part of my learning process in **object-oriented programming, algorithms, and software design**.

---

## 🚀 Features

* ♜ Complete chess board implementation
* ♞ Movement logic for all pieces (King, Queen, Rook, Bishop, Knight, Pawn)
* ♚ Turn-based gameplay
* 📦 Game state management (serialization)
* 🧠 Rule validation (legal moves, captures, etc.)

---

## 🛠️ Technologies Used

* **Java**
* Object-Oriented Programming (OOP)
* File serialization (`.ser`)
* IntelliJ IDEA project structure

---

## 📂 Project Structure

```
src/                 # Main source code
exportToHTML/        # Generated/exported files
.idea/               # IDE configuration
```

---

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/anastasia638/java-chess-game.git
```

2. Open the project in **IntelliJ IDEA** (or any Java IDE)

3. Run the main class:

```bash
Main.java
```
